# Palindrome Assessment

This contains automated tests for the palindrome functionality

## Clone the repo
`git clone https://github.com/azaeng04/palindrome.git`

## Getting set up

Install npm packages:

`npm install`

or

`yarn install`

## Executing the test(s)

`npm test`

or

`yarn test`

## Open up test runner

`npm run cy:open`

or

`yarn cy:open`
